## Weekly Breakdown

### Week 1: Foundations
- DSA: Arrays, Hashing, Two Pointers, Sliding Window, Stack, Linked List (28 problems)
- Build: LRU Cache, Rate Limiter, Consistent Hashing, URL Shortener, Message Queue, Bloom Filter
- DevOps: Docker (Dockerfile, compose, networking, volumes)

### Week 2: Trees, Graphs, Core Patterns
- DSA: Binary Tree, Graphs, Binary Search, Intervals, Heap (30 problems)
- Build: Key-Value Store, Load Balancer, Chat Server, Pub/Sub, API Gateway, Circuit Breaker
- DevOps: Kubernetes (pods, deployments, services, scaling, Helm)
- Agent Course: Google ADK (8 sessions)

### Week 3: DP, Advanced Design, AI/ML
- DSA: Dynamic Programming, Greedy, Backtracking (23 problems)
- Build: Distributed Lock, Search Indexer, Job Scheduler, Task Scheduler, RAG Search, ML Serving, Event Pipeline
- DevOps: Terraform, AWS services, Monitoring, Deployment strategies
- Agent Course: LangChain Deep Agents

### Week 4: Polish, Mock, Apply
- DSA: Revision, weak areas, timed practice (10-15 problems)
- Polish: All READMEs, GitHub CI badges, Dockerfiles + K8s YAMLs
- Mock: 3-4 mock interviews (Pramp)
- Behavioral: 7 STAR stories
- Apply: 20+ companies
